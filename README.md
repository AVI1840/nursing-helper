# סיעוד 360 — מדריך הזכויות המלא לגמלת סיעוד

מדריך אינטראקטיבי לתכנון סל הסיעוד, חישוב שעות והמרה לכסף, וגילוי זכויות נוספות ממשרדים ממשלתיים.

## טכנולוגיה
- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Recharts

## הרצה מקומית
```sh
npm install
npm run dev
```

## בנייה
```sh
npm run build
```

## מבנה תיקיות
```
src/
├── components/   # רכיבי UI
├── data/         # נתונים סטטיים
├── hooks/        # React hooks
├── lib/          # פונקציות עזר
└── pages/        # דפים ראשיים
```
